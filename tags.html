Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header top section of the page used with CSS
div division any section of the page used with CSS
ol ordered list an ordered list that shows numbers
ul unordered list an unordered list that shows points

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>Mary is so wonderful</strong> Mary is so wonderful
a anchor make a hyperlink <a href="https://github.com/LittleFinchy">Github</a> GitHub
img image show an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code make text format as code <code>Mary is the sweetest person</code> Mary is the sweetest person
span span make text format in a span <span>Let's get this bread</span> Let's get this bread
blockquote
 blockquote make text format as blockquote
<blockquote>Let's win this wheat</blockquote>
Let's win this wheat
